## Hva Kan Jeg Gjøre Nå?

Etter restart kan du be meg:

### 1. Sjekk Backend Status

```
Sjekk status på BEFS1 service i Railway
```

### 2. List Environment Variables

```
Vis meg alle environment variables i BEFS1 service
```

### 3. Sett Email Variables

```
Sett RESEND_API_KEY i BEFS1 service til re_xxxxx
```

Eller:

```
Sett følgende environment variables i BEFS1:
- RESEND_API_KEY = re_xxxxx
- EMAIL_FROM = noreply@befs.no
```

### 4. Hent Logs

```
Vis meg siste logs fra BEFS1 service
```

### 5. Sjekk Deploy Status

```
Er BEFS1 service deployet og kjører?
```

## Restart Cursor

**VIKTIG:** Du må restart Cursor for at MCP endringene skal tre i kraft:

1. **Quit Cursor:** `Cmd + Q` (Mac) eller `Alt + F4` (Windows)
2. **Start Cursor på nytt**
3. **Test MCP** ved å be meg sjekke BEFS1 status

---

Sjekk BEFS1 service status og vis meg environment variables
